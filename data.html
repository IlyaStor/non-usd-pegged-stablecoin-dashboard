Data & API

Non-USD pegged stablecoins · 5 tokens · 5 chains · updated hourly

The full dataset behind the solosolver dashboard is public. No key, no signup. Free to use with attribution. If you are a journalist, researcher or developer, this page is for you.

Coverage

  • Networks: Polygon, Stellar, Solana, Tron, Plasma
  • Current tokens: A7A5 (RUB), EURC (EUR), AUD, TRYB (TRY), ARS
  • Metrics: daily transfer count, daily transfer volume (USD)
  • Window: rolling 90 days, refreshed hourly

Schema

data.json is keyed by network. Each network holds newline-separated daily rows; each row is pipe-delimited: date|token|transfer_count|volume_usd.

{
  "success": true,
  "timestamp": "2026-06-09T16:07:34.654Z",
  "data": {
    "tron":    "2026-06-03|A7A5|107|49542559.23\n...",
    "solana":  "2026-06-03|EURC|...|...\n...",
    "polygon": "2026-06-03|AUD|...|...\n...",
    "plasma":  "2026-06-03|TRYB|...|...\n...",
    "stellar": "2026-06-03|ARS|...|...\n..."
  }
}

Quick start

import requests

raw = requests.get("https://analytics.solosolver.xyz/data.json").json()
rows = []
for network, block in raw["data"].items():
    for line in block.strip().splitlines():
        date, token, count, volume = line.split("|")
        rows.append((network, date, token, int(count), float(volume)))

# total 90-day volume for the ruble token A7A5
a7a5 = sum(v for _, _, t, _, v in rows if t == "A7A5")
print(f"A7A5 90d volume: ${a7a5:,.0f}")
